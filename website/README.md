# The Nancy Guthrie Case Archive

A dedicated resource for tracking the timeline, evidence, and developments in the Nancy Guthrie case.

## Objectives
- **Centralize Information**: Provide a single, organized location for all public details.
- **Maintain Accuracy**: Ensure the timeline is verified and updated.
- **Provide Analysis**: Offer clear summaries of complex case elements.

## Site Structure (Planned)
- `/`: Homepage with summary and latest updates.
- `/timeline`: Chronological sequence of events.
- `/evidence`: Categorized details and findings.
- `/about`: Context on the archive's purpose.
